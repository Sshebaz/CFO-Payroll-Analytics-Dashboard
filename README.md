# 📊 CFO Payroll Analytics Dashboard

## 🔍 Overview
This project focuses on analyzing payroll and workforce data to generate business insights using SQL and Power BI. The dashboard provides a comprehensive view of salary distribution, attendance patterns, and departmental expenses to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze payroll data to identify cost trends and workforce patterns  
- Build an interactive dashboard for real-time business insights  
- Support financial decision-making with clear and structured reports  

---

## 🛠️ Technologies Used
- **MySQL** – Data storage, querying, and transformation  
- **Power BI** – Data visualization and dashboard creation  
- **SQL** – Data extraction, joins, aggregations, and optimization  

---

## 📈 Key Features
- Real-time dashboard connected to SQL database  
- KPI tracking for payroll metrics  
- Department-wise expense analysis  
- Attendance and workforce trend analysis  
- Interactive filters and drill-down capabilities  

---

## 🧠 Key Insights
- Identified high-cost departments contributing most to payroll expenses  
- Discovered patterns in employee attendance and compensation  
- Highlighted trends useful for workforce planning and budgeting

## 🚀 How to Use
1. Import dataset into MySQL  
2. Run SQL queries to prepare data  
3. Connect Power BI to MySQL database  
4. Load dashboard file and explore insights  

---

## 📌 Conclusion
This project demonstrates the use of SQL and Power BI to transform raw payroll data into meaningful insights, enabling better financial and workforce decision-making.
